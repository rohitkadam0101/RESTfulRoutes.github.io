RESTful Routes

Table about 7 RESTful routes every backend programmer should know....

Name Path HTTP Verb Purpose Mongoose Method
index /blogs GET List of all blogs Blog.find()
New /blogs/new GET Show new blog form N/A
Create /blogs POST Create new blog and redirects somewhere Blog.create()
Show /blogs/:id GET Show info about one specific blog Blog.findById()
Edit /blogs/:id/edit GET Show edit form for particular blog Blog.findById()
Update /blogs/:id PUT Update a particular blog, then redirect somewhere Blog.findByIdAndUpdate
Destroy /blogs/:id DELETE Delete a particular blog, then redirect somewhere Blog.findByIdAndRemove()
